About

My name is Michael, and this is my website. I’m a Ph.D. student by day/night, and a computer programmer (mostly) by day. I live in Bloomington, Indiana with my wife and two cats. I’m into a lot of things: music theory, computer programming, jazz, hacker culture, text editing, hip culture, Schenkerian theory and analysis, math, and learning new things. You can read about some of the things I’ve done on the projects page.

About this site

This site, www.mcclimon.org, is hosted on Nearly Free Speech, which is a great, cheap web host where you only pay for the bandwidth you use. It’s now a static site generated with jekyll (that seems to be what the hacker community is into these days), after former stints in Ruby, Perl, and PHP. I write primarily in Markdown, which makes it very easy to actually write for the web rather than spending all my time with markup.

The serifed body font on this website is Cardo, a free font from Fonts for Scholars. The sans-serif in the header is PT Sans Narrow, from the folks over at Paratype. Both fonts are hosted by the excellent Google web fonts.

The design inspiration for the site comes primarily from IU’s crimson and cream, which is (sports affiliations aside) a nice color scheme. The grey (gray?) is a neutral background, and the cream container is easier on the eyes than stark white.

I write on my iMac running OS 10.7, though occasionally I’ll use my Toshiba NB305 (a tiny little netbook running Crunchbang Linux at the moment). After brief stints with TextMate, Sublime Text, and Emacs (shudder), I use Vim full time now. (I actually like some things about Emacs, actually...Emacs Lisp is way nicer than Vimscript!) You can see my dotfiles on GitHub, if you're into that sort of thing.
